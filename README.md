# Triple_Clone Coding Assignment - FE

## 💻 프로젝트 실행 방법
1. 프로젝트 클론하거나 다운 받습니다. 

```console
$ git clone https://github.com/zzinao/triple-assignment.git
```

2. 라이브러리를 설치합니다.

```console
$ yarn install
```

3. 프로젝트를 실행합니다. 
```console
$ yarn start
```
